Séance 1 : Introduction à Python, Variables et Types de Données

Objectifs de la Séance

  • Découvrir l'environnement de développement Python.
  • Comprendre les variables et les types de données en Python.
  • Écrire un programme simple qui utilise des variables pour interagir avec l'utilisateur.

Déroulement de la Séance

1. Introduction à l'environnement Python

Présentation de Python :

  • Python est un langage de programmation utilisé pour le développement de jeux vidéo, l'analyse de données et le développement d'applications.
  • Il est simple à apprendre et puissant, adapté aux débutants.

Démonstration :

  • Ouvrir l'environnement de développement (comme IDLE, PyCharm, ou Jupyter Notebook) ou un IDE en ligne comme Replit.
  • Créer un nouveau projet et vérifier que tout fonctionne correctement.

2. Introduction aux Variables

Définition d'une variable :

Une variable est un espace de stockage dans votre programme qui peut contenir des données. C'est comme une boîte pour conserver des informations.

Types de Données :

  • Chaînes de caractères (str) : Texte (ex. : nom = "Mario").
  • Entiers (int) : Nombres entiers (ex. : score = 100).
  • Flottants (float) : Nombres à virgule (ex. : temps = 3.5).
  • Booléens (bool) : Valeurs de vérité (ex. : est_vivant = True).

Exemple de Déclaration de Variables :

                nom = "Mario"
                score = 100
                temps = 3.5
                est_vivant = True

3. Activité Pratique : Écrire un Programme Simple

Objectif de l'activité :

Créer un programme qui demandera le nom et le score d'un utilisateur, puis affichera un message personnalisé.

Instructions :

  • Ouvrir l'IDE et créer un nouveau fichier Python.
  • Écrire le code suivant :
                # Demande à l'utilisateur d'entrer son nom et son score
                nom = input("Quel est votre nom ? ")
                score = input("Quel est votre score ? ")
                
                # Affiche un message de bienvenue avec le nom et le score
                print("Bonjour " + nom + ", votre score est " + score + ".")

Explications du Code :

  • input("Quel est votre nom ?") : Cette fonction attend que l'utilisateur saisisse une réponse. Le texte entre les parenthèses est le message qui s'affiche pour indiquer ce qu'il faut entrer. La valeur saisie sera stockée dans la variable nom.
  • print("Bonjour " + nom + ", votre score est " + score + ".") : Cette ligne affiche un message à l'écran. Le + sert à combiner différentes chaînes de caractères et variables, construisant ainsi une phrase avec les réponses.

4. Testez votre programme :

Lancez votre programme et entrez un nom et un score. Observez la réponse du programme.

5. Personnalisez le programme :

Ajoutez des fonctionnalités, par exemple :

  • Vérifier si le score est un nombre et afficher un message d'erreur s'il ne l'est pas.
  • Inclure des questions supplémentaires, comme « Quel est votre jeu préféré ? » et afficher cette information dans le message final.

Exemple de programme amélioré :

                # Demande à l'utilisateur d'entrer son nom, son score et son jeu préféré
                nom = input("Quel est votre nom ? ")
                score = input("Quel est votre score ? ")
                jeu_prefere = input("Quel est votre jeu préféré ? ")
                
                # Affiche un message personnalisé
                print("Bonjour " + nom + ", votre score est " + score + " et vous aimez jouer à " + jeu_prefere + ".")

6. Discussion et Réflexion

Discussion en groupe :

Réfléchissez à comment utiliser ce programme dans un contexte de jeu vidéo. Par exemple, créer un jeu de rôle qui suit les scores des joueurs et leur fournit des messages personnalisés.

Exemples de mise en contexte :

  • Dans un jeu vidéo, afficher un message comme : "Bravo, Mario ! Vous avez un score de 200 points !"
  • Pour gérer une collection de mangas, imaginer un programme qui suit quels mangas ont été lus et le score de satisfaction pour chacun.

Conclusion :

Cette séance vous a permis de découvrir l'environnement Python, d'utiliser des variables et d'écrire un programme simple qui interagit avec l'utilisateur. Continuez à explorer les possibilités offertes par Python pour aller plus loin.

Séance 9 : Introduction aux Fonctions

Objectifs de la Séance

  • Comprendre l'importance des fonctions dans la programmation.
  • Apprendre à définir et appeler des fonctions en Python.
  • Écrire des programmes simples qui utilisent des fonctions pour organiser le code.

Déroulement de la Séance

1. Introduction aux Fonctions

Qu'est-ce qu'une fonction ?

Une fonction est un bloc de code réutilisable qui effectue une tâche spécifique. Les fonctions permettent de diviser le code en sections plus petites et plus faciles à gérer, d'éviter la répétition de code et de rendre les programmes plus clairs et modulaires.

Pourquoi utiliser des fonctions ?

  • Réutilisabilité : Une fois une fonction définie, elle peut être utilisée autant de fois que nécessaire dans le programme.
  • Clarté : Les fonctions permettent de structurer le code en plusieurs parties logiques.
  • Modularité : Elles facilitent la maintenance du code.

Définir une fonction :

La syntaxe pour définir une fonction est la suivante :


                def nom_de_la_fonction(paramètres):
                    # Bloc d'instructions
                    return valeur_de_retour  # (optionnel)

Exemple de Fonction Simple :


                def saluer():
                    print("Bonjour tout le monde !")

Ici, la fonction saluer() ne prend aucun paramètre et affiche simplement "Bonjour tout le monde !" lorsqu'elle est appelée.

2. Activité Pratique : Écrire des Fonctions Simples

Objectif de l'activité : Créer des fonctions simples et les appeler pour résoudre des problèmes basiques.

Exercice Proposé par Thème :
Thème 1 : Jeux Vidéo - Calcul de Score

Exercice : Créez un programme qui définit une fonction pour calculer le score final d'un joueur en fonction de ses points de base et d'un bonus. Ensuite, appelez cette fonction pour afficher le score final.


                def calculer_score(points, bonus):
                    score_total = points + bonus
                    return score_total
                
                # Demande des points et du bonus
                points = int(input("Entrez vos points : "))
                bonus = int(input("Entrez votre bonus : "))
                
                # Appelle la fonction pour calculer le score
                score_final = calculer_score(points, bonus)
                print("Votre score final est :", score_final)
Thème 2 : Manga - Suivi de Lecture

Exercice : Créez une fonction qui demande à l'utilisateur de saisir un manga et le nombre de chapitres lus, puis affiche le nom du manga et les chapitres lus.


                def suivre_lecture():
                    manga = input("Entrez le nom du manga : ")
                    chapitres = int(input(f"Combien de chapitres avez-vous lus de {manga} ? "))
                    print(f"Vous avez lu {chapitres} chapitres de {manga}.")
                
                # Appel de la fonction
                suivre_lecture()
Thème 3 : Vie Réelle - Calcul de Dépenses

Exercice : Créez une fonction qui calcule le total des dépenses d'une journée. Demandez à l'utilisateur d'entrer les montants de ses dépenses et affichez le total à la fin.


                def calculer_depenses():
                    total = 0
                    for i in range(3):  # Demande 3 montants par exemple
                        depense = float(input(f"Entrez le montant de la dépense {i+1} : "))
                        total += depense
                    return total
                
                # Appel de la fonction pour calculer les dépenses
                total_depenses = calculer_depenses()
                print("Le total des dépenses est :", total_depenses)
Thème 4 : Actualité - Suivi des Événements

Exercice : Créez une fonction qui permet à l'utilisateur d'ajouter des événements d'actualité dans une liste, puis affiche tous les événements ajoutés.


                def ajouter_evenements():
                    evenements = []
                    for i in range(3):  # Par exemple, 3 événements
                        evenement = input(f"Entrez l'événement {i+1} : ")
                        evenements.append(evenement)
                    return evenements
                
                # Appel de la fonction pour ajouter des événements
                evenements = ajouter_evenements()
                print("Les événements saisis sont :")
                for evt in evenements:
                    print("-", evt)

  • Dans un programme de suivi de mangas, une fonction peut être utilisée pour ajouter de nouveaux titres ou calculer la moyenne des chapitres lus.

Objectifs de la Séance

  • Analyser des données sous forme de listes et effectuer des calculs simples comme des moyennes, des totaux et des pourcentages.
  • Comprendre comment représenter les données actuelles en utilisant des outils de programmation.

Déroulement de la Séance

1. Introduction aux Statistiques et à l'Actualité (20 minutes)

Présentation de l'actualité liée au COVID :

Discutez des données disponibles pendant la pandémie, comme le nombre de nouveaux cas, de décès ou de guérisons sur une période donnée (par jour ou par semaine).

Pourquoi analyser des données ?

Analyser des données permet de comprendre les tendances, comme une augmentation ou une diminution des cas, et de prendre des décisions informées.

Données Simplifiées :

Pour cet exercice, les élèves utiliseront une liste de données représentant le nombre de cas quotidiens de COVID sur une semaine donnée.

2. Activité Pratique : Analyse des Cas de COVID sur une Semaine (40 minutes)

Objectif de l'activité :

Écrire un programme qui permet à l'utilisateur de saisir le nombre de nouveaux cas de COVID chaque jour pendant une semaine, puis calculer des statistiques comme la somme totale des cas, la moyenne des cas par jour, et identifier le jour avec le plus grand nombre de cas.

Instructions :

                # Initialisation de la liste pour stocker les cas de chaque jour
                cas_covid = []
                
                # Saisie des cas pour chaque jour de la semaine (7 jours)
                for jour in range(7):
                    cas = int(input(f"Entrez le nombre de cas pour le jour {jour + 1} : "))
                    cas_covid.append(cas)
                
                # Calcul de la somme totale des cas sur la semaine
                total_cas = sum(cas_covid)
                print(f"Nombre total de cas sur la semaine : {total_cas}")
                
                # Calcul de la moyenne des cas par jour
                moyenne_cas = total_cas / len(cas_covid)
                print(f"Moyenne de cas par jour : {moyenne_cas:.2f}")
                
                # Identification du jour avec le plus grand nombre de cas
                jour_max_cas = cas_covid.index(max(cas_covid)) + 1
                print(f"Le jour avec le plus grand nombre de cas est le jour {jour_max_cas} avec {max(cas_covid)} cas.")
2. Testez votre programme :

Saisissez des données fictives pour chaque jour de la semaine et vérifiez que le programme affiche les bonnes statistiques (somme totale, moyenne, jour avec le maximum de cas).

3. Personnalisez votre programme :

Ajoutez des fonctionnalités supplémentaires, comme le calcul du pourcentage d'augmentation ou de diminution des cas par rapport au jour précédent.

3. Exercices Supplémentaires et Discussion (20 minutes)

Exercice Supplémentaire : Tendance des Cas

Modifiez le programme pour qu'il affiche si le nombre de cas a augmenté ou diminué par rapport au jour précédent.

Exemple de Code :

                for jour in range(1, len(cas_covid)):
                    if cas_covid[jour] > cas_covid[jour - 1]:
                        print(f"Le jour {jour + 1}, les cas ont augmenté par rapport au jour précédent.")
                    elif cas_covid[jour] < cas_covid[jour - 1]:
                        print(f"Le jour {jour + 1}, les cas ont diminué par rapport au jour précédent.")
                    else:
                        print(f"Le jour {jour + 1}, le nombre de cas est resté identique.")
Discussion :

Réfléchissez à d'autres statistiques que vous pourriez calculer à partir de ces données. Comment ces informations pourraient-elles être utiles pour les autorités sanitaires ou le public ?
