Cours : Créer un jeu en Python

Objectifs :

  • Comprendre les bases de la programmation Python.
  • Manipuler des variables, des conditions et des boucles.
  • Créer un programme interactif fonctionnel.

1. Introduction

  • Présentation du jeu : "Nous allons coder un jeu où l’ordinateur choisit un nombre secret, et vous devrez le deviner. Il vous dira si votre nombre est trop haut ou trop bas."
  • Concepts abordés :
    • Les variables : pour stocker des informations.
    • Les conditions : pour vérifier si le joueur a trouvé le bon nombre.
    • Les boucles : pour que le joueur puisse continuer à deviner.
    • Les interactions utilisateur : grâce à input().

Avant toute chose, ouvrez le logiciel VSCode, puis créez un fichier : jeuGTN.py

2 : Comprendre les bases du programme

Étape 1 : Initialisation

Explication : On utilise la bibliothèque random pour générer un nombre aléatoire.

    import random

    print("Bienvenue dans 'Devine le nombre' !")
    print("Je pense à un nombre entre 1 et 20. À toi de deviner !")

    # L'ordinateur choisit un nombre aléatoire
    nombre_secret = random.randint(1, 20)

Étape 2 : Ajouter une interaction utilisateur

Explication : On demande au joueur d'entrer un nombre, et le programme affiche ce nombre.

# Demander au joueur d'entrer un nombre
choix = int(input("Entre un nombre : "))
print(f"Tu as choisi : {choix}")

3 : Ajouter de la logique au jeu

Étape 3 : Vérifier si le joueur a trouvé le nombre

Explication : Utilisation des conditions if, elif, et else pour vérifier les réponses.

    if choix < nombre_secret:
        print("Trop bas !")
    elif choix > nombre_secret:
        print("Trop haut !")
    else:
        print("Bravo ! Tu as trouvé le nombre.")

Étape 4 : Répéter avec une boucle

Explication : Une boucle while permet de rejouer jusqu'à trouver le bon nombre.

    tentatives = 0
    devine = False

    while not devine:
        choix = int(input("Entre un nombre : "))
        tentatives += 1

        if choix < nombre_secret:
            print("Trop bas !")
        elif choix > nombre_secret:
            print("Trop haut !")
        else:
            print(f"Bravo ! Tu as trouvé en {tentatives} tentatives.")
            devine = True

4. Activité bonus : Personnalisation

  • Changer la plage du nombre : 
    nombre_secret = random.randint(1, 50)
  • Limiter le nombre d’essais : 
        if tentatives > 5:
        print("Dommage, tu as utilisé tous tes essais !")
        break
  • Ajouter des messages amusants : 
        if choix == 13:
        print("C'est ton chiffre porte-bonheur ? Pas cette fois !")
